HTML

  -HyperText Markup Language

What is HTML?

  • HTML is the standard markup language for creating Web pages.
  • HTML describes the structure of a Web page
  • HTML elements tell the browser how to display the content

HTML Tags

Heading Tags

  • Any document starts with a heading.
  • Headings are titles or subtitles that you want to display on a webpage.
Code 
        <body>
        <h1>This is h1 tag</h1>
        <h2>This is h2 tag</h2>
        <h3>This is h3 tag</h3>
        <h4>This is h4 tag</h4>
        <h5>This is h5 tag</h5>
        <h6>This is h6 tag</h6>
        </body>
Output

This is h1 tag

This is h2 tag

This is h3 tag

This is h4 tag

This is h5 tag
This is h6 tag

Paragraph Tag

The <p> tag offers a way to structure your text into different paragraphs

Code 
        <p>This is the first paragraph of text</p>
        <p>This is the second paragraph of text</p>
        <p>This is the third paragraph of text</p>
Output

This is the first paragraph of text

This is the second paragraph of text

This is the third paragraph of text

Line Break Tag

<br />is an empty tag which follows the text to start from the next line.

Code 
        <p>Hello !!<br />
        Here is a Line Break Tag  <br />
        Thank you, <br />
        Sowmy</p>
Output

Hello !!
Here is a Line Break Tag
Thank you,
Sowmy

Horizontal Line Tag

<hr>defines a thematic break in an HTML page.

Code 
        <p>Hello !!<hr>
        Here is a Horizontal Line Tag  <hr>
        Thank you, <hr>
        Sowmy</p>
Output

Hello !!

Here is a Horizontal Line Tag
Thank you,
Sowmy

Block Quote Tag

<blockquote> tag specifies a section that is quoted from another source.

Code 
            <blockquote>
                <p>"Be yourself; everyone else is already taken." - Oscar Wilde</p>
            </blockquote>
Output

"Be yourself; everyone else is already taken." - Oscar Wilde

HTML Elements

Bold Italic & Underline Elements

  • <b> bolds the text

  • <i> makes the text italic

  • <u> underlines the text

Code 
            <p>This Text is <b>BOLD</b><br />
            <p>This Text is <i>Italic</i><br />
            <p>This Text is <u>Underlined</u><p>
Output 
            
This Text is BOLD
This Text is Italic
This Text is Underlined

Superscript & Subscript Elements

  • <sup> defines superscript text

  • <sub> defines subscript text

Code 
            <p>This Text is <sup>Supersripted</sup><br />
            This Text is <sub>Subscripted</sub></p>
Output 
            
This Text is Superscripted
This Text is Subscripted

HTML Lists

Unordered List

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag. The list items will be marked with bullets by default

Code 
            <ul>
            <li> Item 1 </li>
            <li> Item 2 </li>
            </ul>
Output 
            
    
            
  •  Item 1 
    • 
            
  •  Item 2 
    •

Ordered List

An ordered list starts with the <ul> tag. Each list item starts with the <li> tag. The list items will be marked with Numbers by default

Code 
            <ol>
            <li> Item 1 </li>
            <li> Item 2 </li>
            </ol>
Output 
            
    
            
  1.  Item 1 
    2. 
            
  2.  Item 2 
    3.

Marquee Tag

The <marquee> tag is a container tag of HTML is implemented for creating scrollable text or images within a web page from either left to right or vice versa, or top to bottom or vice versa.

Code 
            <marquee>This Text is BOLD</marquee>
            <marquee behavior="scroll" direction="left" scrollamount="5">Slow Scrolling</marquee>
            <marquee behavior="scroll" direction="right" scrollamount="12">Little Fast Scrolling</marquee>
            <marquee behavior="scroll" direction="left" scrollamount="20">Fast Scrolling</marquee>
            <marquee behavior="scroll" direction="right" scrollamount="50">Very Fast Scrolling</marquee>
Output 
            Slow Scrolling
            Little Fast Scrolling
            Fast Scrolling
            Very Fast Scrolling
